# OCDWP3
## Backend
### Backend uses PORT=5678
### Install dependencies:
```zsh
cd Backend
nvm install 18
nvm use 18
npm install
```
### Run the backend:
```zsh
npm start
```
## FrontEnd
### Install dependencies:
```zsh
cd frontend
nvm install 22
nvm use 22
npm install
npm start
```
## JSDoc
```zsh
npm run docs
```
